hexo+github建立个人博客总结

发表于 2018-01-29 | 更新于: 2018-01-29 | 分类于 Hexo相关文档 | 浏览 次

环境准备：

  • 域名
  • nodejs
  • git
  • GitHub pages
  • 百度统计
  • 七牛云存储配置图床
  • 链接的社交账号
  • 添加评论：多说

操作步骤：

  • 本地安装git，注册github账号
  • 安装nodejs(npm也连带安装)
  • 安装并初始化hexo：
  • cnpm install hexo -g #安装
  • hexo init #初始化
  • hexo n “博客名称” #新建博客文章
  • hexo g #博客生成
  • hexo d #博客发布
